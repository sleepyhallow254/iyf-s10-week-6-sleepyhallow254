## Project Description
This project is a Weather Dashboard application that uses the OpenWeatherMap API to fetch and display real-time weather data. Users can search for any city and view current weather conditions along with a 5-day forecast. The project demonstrates working with APIs, asynchronous JavaScript, and dynamic DOM manipulation.

---

## Technologies Used
- HTML5  
- CSS3  
- JavaScript  
- OpenWeatherMap API  

---

## Features
- Search for weather by city name  
- Display current weather conditions  
- Show temperature, humidity, wind speed, and pressure  
- Display weather icon and description  
- 5-day weather forecast  
- Error handling for invalid cities  
- Loading state while fetching data  
- Save and display recent searches (localStorage)  
- Click recent searches to search again
